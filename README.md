本软件是一个gui密码算法工具. 
基于[gmssl](https://github.com/guanzhi/GmSSL) , 支持sm2/3/4和des/aes/rsa算法. 

- 所有数据均为16进制.
- 使用作为界面库, 在高分屏上支持良好.
- 自带一些字符处理功能
